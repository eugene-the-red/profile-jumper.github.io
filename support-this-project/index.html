Support This Project

How can you support this project? Would you like to contribute by helping with anything to benefit this project, see more below on how to support this project.

First things first!

As this is an open souce project, the benevolant dictator of this project has decided not to accept your money! Yes, we don't want your money! If you really like this browser extension, it would be really marvelous if you could simply tell someone else about it, that's right, just tell someone about it and it will make us happy.

How can you contribute?

  • Spread the word
  • We need translators
  • See a bug, help us fix it
  • Suggest a new feature

How do you benefit?

We really want to recognize all contributors and on the credits section, your profile will be added there. This will happen in chronological order. In the credits section you can link to your online profiles or a project you are working on. Not just that, you can increase you reputation and use this as social credit and even state this on your CV/Resume!
